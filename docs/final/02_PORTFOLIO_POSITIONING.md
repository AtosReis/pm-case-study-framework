# Portfolio Positioning

## PM Framework + LLM-Assisted Product Audit

This project is not an AI PRD generator.

It is a Product Manager operating framework with an optional LLM-assisted audit layer.

The framework helps structure the first product panorama of an ambiguous project:

- business goal;
- actors;
- systems;
- user journey;
- product decisions;
- MVP scope;
- use cases;
- PRD;
- engineering tickets;
- risks;
- success metrics.

The LLM layer then reviews that first panorama and identifies weak points before the work reaches engineering or stakeholders.

## What This Project Does Not Claim

This project does not claim that I became a senior Product Manager overnight.

It does not claim that AI can replace product judgment.

It does not claim that an LLM should make product decisions.

It does not claim that one generated PRD is automatically production-ready.

The project is intentionally positioned as a learning artifact, a working framework, and a lightweight product tool.

---

## The Core Lesson

The original mistake was moving too quickly from business request to AI/automation.

The corrected process is:

1. Understand the business goal.
2. Identify users and actors.
3. Map systems and ownership.
4. Clarify the user journey.
5. Make product decisions explicit.
6. Define MVP scope.
7. Write use cases.
8. Create a PRD.
9. Translate requirements into engineering tickets.
10. Use LLM review to audit the first draft and expose weak points.

The LLM is not used to skip PM thinking.

The LLM is used to review PM thinking.

---

## How the LLM Layer Is Used

The LLM review layer is designed to identify:

- missing product decisions;
- unclear assumptions;
- weak acceptance criteria;
- scope risks;
- technical dependencies;
- data ownership questions;
- legal/compliance questions;
- follow-up discovery questions;
- recommended improvements.

This is useful because early product work often looks complete before it is actually complete.

The LLM acts as a critical reviewer that helps expose what still needs clarification.
